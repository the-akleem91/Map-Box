## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Install the Expo app from Playstore or Open in Emulator

3. Start the app

   ```bash
    npx expo start
   ```

4. Switch to expo on CLI by Pressing `s`

5. Scan the QR code from the Expo App
